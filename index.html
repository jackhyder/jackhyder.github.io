Projects

  • Stop Clickbait Brower Extension

    Stop Clickbait! is an open-source project to build a browser extension to fight clickbait on social media, using HTML, CSS, and JavaScript on the Frontend and mostly PHP and JavaScript on the Backend. Clickbait is a technique that uses exaggerated or misleading headlines to pull users into reading articles, often with the intent to push users through many pages to get to the heart of the actual content. Because of the current model of web-based advertising, this brings in a lot of revenue for the website hosts without providing meaningful content to the users. Our extension will allow users to flag and report these parasitic articles to allow for community-sourced curation of media. Expect a launch sometime in the second half of 2017!

  • www.InterviewMath.com

    This website was built for a client to serve as a supplement to the book of the same name. It allows users to practice timing themselves against real-world math-based interview questions from the book. The website uses JavaScript, HTML, and CSS, and includes jQuery, Google Analytics, and BootStrap as well.

On a team of 6 people, we built a full stack application that tracked the responsiveness of servers in different geographical locations. The application was also our introduction to using version control in the form of git and GitHub, and was built using the MEAN stack and Google Maps API.
